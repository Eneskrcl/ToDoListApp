# ToDoListApp

Basit bir Python To-Do List uygulaması. Kullanıcılar görev ekleyebilir, silebilir ve mevcut görevleri listeleyebilir. Görevler SQLite veritabanında saklanır.

## Gereksinimler

- Python 3.x
- SQLite (Python ile birlikte gelir)
- Ek paket gerekmez (standart kütüphaneler kullanılmıştır)
